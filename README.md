# 마법사의 서고

> 독서노트를 작성하듯이 영상물에 대해서도 기록을 하면 어떨까에서 시작했다.

## 소개

**마법사의 서고**는 영화와 드라마 작품에 대한 나만의 질문과 답변을 기록하는 서비스입니다. 독서노트를 작성하듯이, 영상물을 감상한 후 깊이 있는 사고와 기록을 남길 수 있는 공간을 제공합니다.

## 주요 기능

### 📚 작품 검색 및 추가
- TMDB API를 활용한 영화/드라마 검색
- 작품을 선택하면 서재에 책 형태로 추가됨
- 각 작품은 질문과 답변의 기록 공간이 됨

### ✨ AI 기반 질문 생성
- 작품의 줄거리를 바탕으로 AI가 질문을 자동 생성
- 기존 답변을 참고하여 더 깊이 있는 후속 질문 생성
- 객관식 또는 주관식 질문 지원

### 📝 질문과 답변 기록
- 각 작품에 대해 여러 질문과 답변을 기록 가능
- 질문 직접 작성 기능
- 질문 삭제 및 수정 기능

### 🎨 아름다운 서재 UI
- 고대 도서관을 연상시키는 3단 선반 레이아웃
- 책 형태의 인터페이스로 작품을 시각화
- 질문 수에 따라 책의 두께가 자동 조정

### 💾 로컬 스토리지 저장
- 브라우저 로컬 스토리지에 모든 기록 저장
- 언제든지 이전 기록 확인 및 수정 가능

### 📤 공유 기능
- 카카오톡으로 질문과 답변 공유
- 다른 사용자들이 공유한 기록 확인 가능
- Vercel KV를 활용한 공유 기록 저장

## 시작하기

1. 작품 검색: 상단의 "+" 버튼을 클릭하여 원하는 영화나 드라마를 검색합니다.
2. 작품 선택: 검색 결과에서 원하는 작품을 선택하면 서재에 추가됩니다.
3. 질문 생성: 책을 클릭하여 열고, "새 질문 연성" 버튼으로 AI가 생성한 질문을 받거나 직접 질문을 작성합니다.
4. 답변 기록: 각 질문에 대한 답변을 작성하고 저장합니다.
5. 공유하기: 마음에 드는 질문과 답변을 카카오톡으로 공유하거나, 전체 책을 공유 기록에 저장할 수 있습니다.

## 기술 스택

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **API**: TMDB API (영화/드라마 검색), OpenAI API (질문 생성)
- **Storage**: LocalStorage (로컬 저장), Vercel KV (공유 기록)
- **Deployment**: Vercel

## 프로젝트 배경

독서노트를 작성하듯이 영상물에 대해서도 기록을 하면 어떨까에서 시작했습니다. 단순히 "재미있었다"를 넘어서, 작품에 대해 더 깊이 생각하고 그 생각을 기록으로 남기는 것이 목표입니다.

영화나 드라마를 보고 나서 떠오르는 질문들 - "이 인물의 선택이 옳았을까?", "이 장면의 의미는 무엇일까?", "내가 이 상황이라면 어떻게 했을까?" - 이런 질문들에 대한 답변을 기록하며, 작품에 대한 이해를 깊게 할 수 있습니다.
