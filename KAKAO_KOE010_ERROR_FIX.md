# KOE010 에러 해결 가이드

## 에러 내용
- **에러 코드**: KOE010
- **에러 메시지**: invalid_client (Bad client credentials)
- **의미**: REST API 키가 잘못되었거나 카카오 개발자 콘솔 설정 문제

## 스크린샷 확인 결과

✅ **Redirect URI 설정**: 올바르게 설정됨
- `http://localhost:3000` ✅
- `https://magic-house-chi.vercel.app` ✅

## 확인해야 할 사항

### 1. REST API 키 확인 (가장 중요!)

스크린샷에서 보이는 페이지는 **REST API 키 설정 페이지**입니다.

1. 이 페이지에서 **"키 이름"** 옆에 있는 **REST API 키 값**을 확인하세요
2. 현재 `.env` 파일에 설정된 값: `bec86a0152235484912ff8cc174fa8da`
3. 카카오 개발자 콘솔에서 보이는 REST API 키와 **정확히 일치**하는지 확인

**중요**: 
- JavaScript 키 (`565745e8bea5623d9bb958af626f4cb4`)와 REST API 키는 **다릅니다**!
- 스크린샷에서 보이는 키 이름 "Default Rest API Key" 옆에 실제 키 값이 있을 것입니다
- 그 값을 복사해서 `.env` 파일에 정확히 입력해야 합니다

### 2. 호출 허용 IP 주소 확인

스크린샷에서 "호출 허용 IP 주소" 섹션이 비어있는 것으로 보입니다. ✅

**로컬 개발 환경에서는 비어있어야 합니다!**
- IP 주소를 설정하면 해당 IP에서만 호출 가능
- 로컬 개발 시 IP를 설정하면 문제가 발생할 수 있음
- 현재 비어있으므로 문제 없음 ✅

### 3. REST API 키 찾는 방법

스크린샷에서 보이는 페이지가 아니라:

1. **앱 설정** → **앱 키** 메뉴로 이동
2. 거기서 **REST API 키**를 확인
3. 그 값을 복사

또는 현재 페이지에서:
- "Default Rest API Key" 옆에 실제 키 값이 표시되어 있을 수 있습니다
- 그 값을 확인하세요

## 해결 단계

1. **카카오 개발자 콘솔** → **앱 설정** → **앱 키** 이동
2. **REST API 키** 값 확인 및 복사
3. `.env` 파일의 `KAKAO_REST_API_KEY` 값과 비교
4. 다르면 `.env` 파일 수정
5. 서버 재시작
6. 다시 테스트
